== Description ==

**Simple Custom SEO** is a ultra-lightweight plugin that lets you manually set a custom SEO title and meta description for each post and page.

Perfect for developers, lean websites, or those who don’t want bloated SEO plugins. Built for speed and flexibility.

**Features:**
- Add a custom SEO title per post/page
- Add a custom meta description (auto-trimmed to 160 characters on output)
- Displays in the `<head>` section of your site's front end
- No bloat, no admin clutter, no unnecessary settings

Designed for high-performance WordPress sites that want SEO control without the weight of plugins like Yoast or RankMath.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/simple-custom-seo`
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Edit any post or page and look for the **Custom SEO Settings** meta box
4. Enter your SEO title and meta description
5. Save or update the post — done!

== Frequently Asked Questions ==

= Will this conflict with other SEO plugins? =
If you're using Yoast, RankMath, or similar, they may override the output from this plugin. Use only one SEO plugin at a time for title/description control.

= Does it support custom post types? =
Currently supports posts and pages. Custom post type support can be added easily — let us know if you'd like that!

= Will this slow down my site? =
Not at all. This plugin is minimal and lightweight. It won’t load any unnecessary assets or code.
